## 1.0.0 (Unreleased)

### Features

- Initial release
- Faithful recreation of the 1972 Atari Pong arcade game
- Single-player mode with AI opponent (Easy / Normal / Hard)
- Two-player local mode (same keyboard)
- Terminal beep sound effects (with --no-sound flag)
- Configurable winning score (default: 11)
- Real-time terminal rendering at 30 FPS
- ASCII art title screen
- Monochrome black-and-white aesthetic
